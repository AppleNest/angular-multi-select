Grouping

Grouping is done by nested additional arrays of objects in your input-model

First, in your directive declaration, set a grouping marker using the group-property, let's say group-property="sub". This is our group marker. Now, in your controller, grouping is made by nesting some objects in your input-model array with this sub key. Example: 

$scope.data = [
    {   
        name: 'People',
        sub: [
            {
                name: 'Person A'
            },
            {
                name: 'Person B'
            }
        ]
    },
    {
        name: 'Dogs',
        sub: [
            {
                name: 'Dog A'
            },
            {
                name: 'Dob B'
            }
        ]
    }
];

The directive will take care of indentation of grouped items. It supports infinite nested grouping. You can use HTML tags on groups as usual. Clicking the group header will select/deselect all visible items under that particular group EXCEPT on single selection-mode, in which case it will deselect everything, then try to select the group if that group contains only 1 sub-item.

    output-model="outputBrowsers"
    item-label="{| icon |} {| name |}"
    tick-property="checked"
    group-property="sub"
>
</div>
