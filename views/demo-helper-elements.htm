Demo > Helper Elements

You can define which helper elements to be displayed by using attribute helper-elements, such as:

 

  • View
  • Controller

Define your directive:

<div
    angular-multi-select
    input-model="webBrowsersGrouped"
    output-model="outputBrowsers"
    item-label="{{ icon }} {{ name }}"
    tick-property="checked"
    ...
    helper-elements="all none noreset nofilter"
>
</div>

Define your input-model:

$scope.outputBrowsers = [
  { icon: "{{removeHost(row.icon)}}", name: "{{row.name}}", maker: "{{row.maker}}", ticked: {{row.ticked}} },
];
Other examples

The attribute accepts a combination of "all none filter reset", in space separated string. Example:

  • helper-elements="filter": will display the search (filter) field only
  • helper-elements="all none": will display the "Select all" and "Select none" buttons
  • helper-elements="xyz": will display no helper elements at all
  • helper-elements="": will display "Reset" and "Search" helper elements if in single mode or everything if in multi mode.
