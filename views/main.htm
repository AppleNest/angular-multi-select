AngularJs MultiSelect

AngularJS directive which creates a dropdown widget with multiple or single selection mode.


Features
  • Can be easily combined with other Javascript libraries such as Twitter Bootstrap.
  • Multiple or single selection mode
  • Use HTML & CSS in the button itself and the items labels
  • Unlimited nested grouping
  • Keyboard navigation support
  • Event callbacks available
  • Directive methods callable from the controller via a nice API
  • Configurable
  • i18n. Easily translated to different languages.
